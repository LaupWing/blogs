# Blog Structure

This is a reference, not a rulebook. Claude should read the draft first and use this as a lens — not a template to force onto the content.

## General Flow

1. **Hook** — the opening. See HOOK.md. This is the most important part.
2. **Problem / Context** — why does this matter? What's the situation?
3. **Body** — the substance. Can be tips, a story, an argument, a how-to.
4. **Takeaway** — what should the reader walk away with? Keep it concrete.

## Tone

- Conversational, direct, no fluff
- Write like you're explaining to a smart friend
- Short sentences. Short paragraphs.
- No corporate language

## Length

- No fixed length — as long as it needs to be, no longer
- If a section feels forced, cut it
